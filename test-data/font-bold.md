## Change the size of text in Windows 10
Change the size of text in Windows 10 using Display settings, zoom in or out in Microsoft Edge, or use Magnified.

**In Windows**: Select the **Start** Windows logo Start button menu, then type **change display settings** in the search box on the taskbar. From the list of results, choose **Change display settings** and then choose the size under **Scale and layout**.
![Display](http://i0.kym-cdn.com/entries/icons/mobile/000/005/608/nyan-cat-01-625x450.jpg)
**In Microsoft Edge**: Select **More actions** Three-dot More icon, and then zoom in or out.

**Use Magnifier**: Open Magnifier to make the part of any screen larger. Press the **Windows logo key** Windows logo Start button + **Plus** Plus-sign Add icon.
